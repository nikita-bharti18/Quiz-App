# Quiz App in Java

A simple console-based Quiz Application built using Java.

## Features
- Multiple-choice questions
- Score calculation
- Beginner-friendly project
- Clean code structure

## Technologies Used
- Java
- OOP Concepts
- ArrayList

## How to Run

1. Compile the files
```bash
javac Main.java Question.java
```
2. Run the program
```
java Main
```
## Project Output

- Displays questions
- Takes user input
- Shows final score
